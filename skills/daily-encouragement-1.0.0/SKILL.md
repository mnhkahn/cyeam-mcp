# 每日鼓励 (Daily Encouragement)

查询今天是节日还是工作日，生成一句包含状态识别、鼓励话语和鸡汤语录的完整句子。

## 使用方式

告诉你的AI助手：
> 给我今天的鼓励
> 今天是什么日子？

## 步骤

1. 调用节假日 API 判断今天的状态
2. 根据结果选择鼓励方向：
   - 节假日/周末 → 鼓励好好玩 🎉
   - 工作日 → 鼓励好好干 💪
3. 加上一句鸡汤语录
4. 输出**一句完整的话**

## 节假日查询 API

免费接口，无需密钥：

```bash
curl -s "https://timor.tech/api/holiday/info/"
```

在路径末尾追加今天的日期，例如 `https://timor.tech/api/holiday/info/2026-06-04`。

### API 返回结构

```json
{
  "code": 0,
  "type": {
    "type": 0,
    "name": "周四",
    "week": 4
  },
  "holiday": null
}
```

### type.type 取值说明

| 值 | 含义 | 鼓励方向 |
|----|------|----------|
| 0  | 工作日（上班） | 💪 好好干 |
| 1  | 法定节假日（放假） | 🎉 好好玩 |
| 2  | 调休上班日 | 💪 好好干 |
| 3  | 周末 | 🎉 好好玩 |

### holiday 字段

当 `holiday` 不为 `null` 时，包含节假日名称等信息，应在生成句子时提及该节日名称。

## 示例输出

**工作日示例：**
> 今天是周四，虽然是个普通工作日，但认真工作的你特别酷 💪 种一棵树最好的时间是十年前，其次是现在。——加油！

**节假日示例：**
> 今天是端午节，放假啦！好好享受假期吧 🎉 生活不止眼前的苟且，还有诗和远方。——去探索吧！

## 注意

- 只输出**一句话**，不要分段
- 不要用"哈哈"之类的无意义填充词
- 输出风格：自然、真诚、有力量
